## 🚀 Next Steps: Push to GitHub (Simple!)

### Option 1: Using Web Interface (Easiest)
1. Go to [github.com/new](https://github.com/new)
2. Create a **new repository** named `portfolio-ashok`
3. Copy the push commands GitHub gives you
4. Paste them in your terminal

### Option 2: Command Line (Quick Reference)

```bash
# Open terminal and navigate to portfolio folder
cd "/Users/sudiprajmandal/Downloads/Downloads/Ashok/untitled folder"

# Add GitHub remote (replace with YOUR GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/portfolio-ashok.git

# Rename master → main
git branch -m master main

# Push to GitHub
git push -u origin main
```

### Step 3: Enable GitHub Pages (Automatic)
1. Go to your repo → **Settings** → **Pages**
2. Select **"Deploy from a branch"** with **"main"** branch
3. Click Save
4. Done! ✅ Your portfolio is LIVE in 2-3 minutes
